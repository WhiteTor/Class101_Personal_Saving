기능: 블럭 이동 및 합치기 기능
담당: 김민우

-------------기능설명-------------

function 위로 이동 ()
  모든 개체를 위로 정렬
if 상하로 맞닿은 블록의 값이 동일
  합쳐서 위로 올림

function 아래로 이동 ()
모든 개체를 아래로 정렬
if 상하로 맞닿은 블록의 값이 동일
  합쳐서 아래로 내림

function 오른쪽 이동 ()
모든 개체를 오른쪽으로 정렬
if 좌우로 맞닿은 블록의 값이 동일
  합쳐서 오른쪽으로 밀기

function 왼쪽 이동 ()
모든 개체를 왼쪽으로 정렬
if 좌우로 맞닿은 블록의 값이 동일
  합쳐서 왼쪽으로 밀기
