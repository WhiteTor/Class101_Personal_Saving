상하좌우 이동+합치기
function 위로 이동 ()
  모든 개체를 위로 정렬
if 상하로 맞닿은 블록의 값이 동일
  합쳐서 위로 올림

function 아래로 이동 ()
모든 개체를 아래로 정렬
if 상하로 맞닿은 블록의 값이 동일
  합쳐서 아래로 내림

function 오른쪽 이동 ()
모든 개체를 오른쪽으로 정렬
if 좌우로 맞닿은 블록의 값이 동일
  합쳐서 오른쪽으로 밀기

function 왼쪽 이동 ()
모든 개체를 왼쪽으로 정렬
if 좌우로 맞닿은 블록의 값이 동일
  합쳐서 왼쪽으로 밀기

새로운 블럭생성
- 게임 시작시 2 또는 4인 블럭 2개를 빈 공간에 생성시킨다. 사용자가 블럭 이동을 한 뒤, 빈 공간에 2 또는 4인 블럭을 1개 생성한다. 이때 2가 생성될 확률은 90%, 4가 생성될 확률은 10%로 한다.

점수
-합치기를 한 후에 새로 만들어진 숫자만큼 점수를 더한다.

언두
- 진행 상태를 stack에 저장 
  단, 지속적인 Undo를 사용한다면 게임에 재미요소를 낮출 수 있고, 메모리 할당도 생각하여 가장 최근 3번 정도의 진행 상황만 저장.

나가기
-if문을 통해 방향키가 아닌 특정 키를 입력한다면 break 하여 나간다.
